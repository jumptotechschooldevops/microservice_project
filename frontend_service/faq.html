📚 Courses
Absolutely no prior Linux experience is required. The Fundamentals course starts from the very beginning — we'll explain what Linux is, why it matters, and walk you through your very first command. All you need is a computer with a modern web browser. We provide the Linux environment in-browser so there's nothing to install.
All course purchases include lifetime access. Once you buy a course, you own it forever — including all future updates, new lessons, and additional labs we add to that course. We update our courses regularly to keep pace with new Linux kernel versions and industry practices.
All courses are 100% self-paced. You can start, pause, and resume at any time. There are no deadlines or expiry dates. That said, we recommend dedicating at least 5-7 hours per week for the best learning momentum. We also run optional live Q&A sessions every month for enrolled students.
If you're new to Linux, always start with Linux Fundamentals. From there, the recommended path is: Fundamentals → System Administration → Shell Scripting → Networking & Security → Linux for DevOps → RHCSA Prep. Each course builds on the previous one, though experienced users can jump in at any level.

💻 Technical Requirements
The only requirement is a modern web browser (Chrome, Firefox, Safari, or Edge) with a stable internet connection. Our browser-based lab environment provides a real Linux terminal — no virtual machines, no downloads, no local setup required. For advanced courses, we optionally recommend 8GB RAM if you want to run local VMs alongside the course.
We primarily use Ubuntu LTS and Rocky Linux (RHEL-compatible) in our labs. Most concepts and commands are cross-distro, but we note any significant differences between Debian and RHEL families throughout the courses.
Lab environments are available for the duration of each lesson session and reset between exercises by design — this ensures you always start from a clean state. For long-running projects, the System Administration and DevOps courses provide persistent environments you can save and return to.
